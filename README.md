# 💼 JobTrack — Job Application Tracker

> A modern, polished job application tracker built with React. Track applications, visualize your response rate, and manage your job search — all saved locally in your browser.

## 🔗 Links
- 🌐 **Live Demo:** [harjith2004.github.io/jobtrack](https://harjith2004.github.io/jobtrack)
- 💻 **GitHub:** [github.com/Harjith2004/jobtrack](https://github.com/Harjith2004/jobtrack)

## ✨ Features
- 📊 Dashboard with animated response-rate ring
- 📋 Add, edit, delete job applications
- 🔍 Search and filter by status
- 📈 Analytics with pie chart and bar chart
- 💾 Data saved automatically in browser (localStorage)
- 🎨 Modern dark UI with custom design system

## 🛠️ Tech Stack
- React 18
- Recharts (data visualization)
- Plus Jakarta Sans + Inter (typography)
- localStorage (no backend needed)
